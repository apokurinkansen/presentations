# 事例2: ハウステンボス — 発表者メモ

## 基本情報

- 発生日: 2025年8月29日
- 公表日: 2025年8月31日（第一報）、2025年12月12日（調査結果）
- 攻撃種別: ランサムウェア

## 攻撃の詳細

### 侵入経路

- ハウステンボスが利用していた**リモートアクセス機器**を経由して第三者がネットワークに不正侵入
- 業務管理システム等のサーバーの一部でファイルが暗号化された

### 時系列

| 日付 | 出来事 |
|------|--------|
| 2025年8月29日 | ファイル暗号化を検知 |
| 同日 | サーバー停止・ネットワーク遮断を実施 |
| 同日 | 個人情報保護委員会・警察に報告 |
| 2025年8月31日 | 第一報を公表 |
| 2025年10月1日 | システム復旧完了 |
| 2025年12月12日 | 調査結果を公表 |

### サービスへの影響（復旧まで約1ヶ月）

- 公式アプリのアトラクション待ち時間表示が休止
- 一部レストランのレシート発行不可
- 発注システムの利用制限

## 漏洩した情報の詳細（約154万6,000人分）

### 対象者の内訳

| 対象者 | 人数 | 漏洩情報 |
|--------|------|---------|
| 顧客 | 約149万9,300人 | 氏名、生年月日、性別、住所、電話番号、メールアドレス |
| 役職員・家族（退職者含む） | 約3万7,300人 | 上記に加え、**マイナンバー**、健康診断結果、障がいに関する情報 |
| 取引先 | 約9,400人 | 氏名、社名、住所、電話番号、メールアドレス、**マイナンバー** |

### 漏洩しなかった情報

- クレジットカード情報（そもそも保有していない）

### 悪用被害

- 2025年12月12日時点で、流出した情報が第三者に悪用された形跡は確認されていない

## 研修で強調すべきポイント

- **リモートアクセス機器が侵入口**: VPN やリモートデスクトップ機器の脆弱性は近年のランサムウェアの主要な侵入経路
- **マイナンバーも漏洩対象**: 役職員・取引先のマイナンバーが含まれており、特に深刻
- **初動対応の速さ**: 検知当日にサーバー停止・ネットワーク遮断・当局報告を実施（良い対応例）
- **KADOKAWA との比較**: 同じランサムウェアでも侵入経路が異なる（フィッシング vs リモートアクセス機器）
