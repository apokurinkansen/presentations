# 事例1: KADOKAWA — 発表者メモ

## 基本情報

- 発生日: 2024年6月8日
- 公表日: 2024年8月5日（第一報は6月14日）
- 攻撃種別: ランサムウェア

## 攻撃の詳細

### 侵入経路

- フィッシング等により従業員のアカウント情報が窃取された
- 窃取されたアカウントで社内ネットワークに侵入
- ランサムウェアを含むマルウェアが実行され、データ窃取と暗号化が行われた

### 影響を受けたサービス

- ニコニコ動画、ニコニコ生放送などニコニコファミリーサービス全般
- KADOKAWA グループの複数の Web サイト
- 出版事業の基幹システム（出荷量が一時的に平常時の1/3に低下）

## 漏洩した情報の詳細（254,241人分）

### 対象者の内訳

| 対象者 | 漏洩情報 |
|--------|---------|
| N/S高・専門学校の在校生・卒業生・保護者・出願者 | 氏名、生年月日、住所、電話番号、メールアドレス、活動名、口座情報など |
| ドワンゴ従業員（派遣・契約含む） | 氏名、生年月日、住所、電話番号、メールアドレスなど |
| クリエイター・個人事業主 | 氏名、住所、電話番号、メールアドレスなど |
| 取引先の関係者 | 氏名、メールアドレスなど |
| 面接参加者 | 氏名、メールアドレスなど |

### 漏洩しなかった情報

- ニコニコユーザーのアカウント情報（ログインメールアドレス、パスワード）
- クレジットカード情報

## 被害の規模

| 項目 | 内容 |
|------|------|
| 個人情報漏洩 | 254,241人分 |
| サービス停止 | 約3ヶ月（6月8日〜9月以降段階的に復旧） |
| 特別損失 | 36億円（復旧費用・クリエイター補償費用等） |
| 売上影響 | 売上高 ▲84億円、営業利益 ▲64億円（2025年3月期） |

## 研修で強調すべきポイント

- **フィッシングが起点**: 高度な技術的攻撃ではなく、従業員のアカウント窃取が入口
- **被害の連鎖**: 侵入→潜伏→横展開→暗号化と、一つの侵入から被害が拡大
- **金銭的インパクト**: 36億円の特別損失に加え、売上も84億円減少
- **身代金を払っても解決しない**: 一部報道では身代金を支払ったとされるが、データは復旧されなかった（※公式未確認情報）
